## ✦ About

**Nord 0** is a lightweight UI framework designed around one idea:

> *Interfaces should be predictable, beautiful, and effortless to scale.*

It's not trying to be everything. It's trying to be **correct** — a small, well-considered set of primitives that stay out of your way and never surprise you.

## ✦ Why Nord 0

Most UI frameworks grow feature by feature until the defaults contradict each other. Nord 0 takes the opposite approach — every primitive is derived from the same small set of rules, so the system stays predictable no matter how large your interface gets.

<br>

## ✦ Getting Started

Drop this into your script — no installer, no build step, no dependencies:

```lua
local Nord = loadstring(game:HttpGet("https://raw.githubusercontent.com/roguerousseau/Nord-0/refs/heads/main/src.lua"))()
```

One line, and the framework is live in your environment. From here, check the [documentation](https://rousseau.gitbook.io/nord-docs/) for how to build your first window, or browse working [examples](https://github.com/roguerousseau/Nord-0/tree/main/Examples) to see Nord 0 in practice.
