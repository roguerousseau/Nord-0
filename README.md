## About

**Nord 0** is a lightweight UI framework designed around one idea:

> Interfaces should be predictable, beautiful, and effortless to scale.

**Nord 0** focuses on:

- **Clarity** — every component has a purpose.
- **Consistency** — one system, infinite possibilities.
- **Precision** — carefully crafted defaults.
- **Performance** — minimal overhead, maximum output.
